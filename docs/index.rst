=================
construct_houdini
=================

Side Effects Houdini integration for Construct. Initializes Construct when
Houdini launches and provides load, save and publish action tasks for various
file types.


Installation
============

.. code-block:: console

    $ pip install git+ssh://git@github.com/construct-org/construct_houdini.git
